## 一、網站定位

提供大安籃球聯盟成員一個**每週查閱賽事資訊的核心平台**，取代現有 Google Sheet，讓賽程、戰績、工作人員資訊能以清晰、美觀、手機友善的方式呈現。

## 三、各頁面詳細內容規劃

---

### 🏠 首頁

**目的：** 讓成員一眼掌握最重要的當週資訊，快速導向所需頁面。

**內容區塊：**

- **聯盟標題橫幅**
  - 屆次名稱（第25屆大安ㄍㄤㄍㄤ好籃球聯盟）
  - 賽季進度（例：例行賽 第3週 / 共10週）

- **下次比賽倒數**
  - 距離下場比賽的倒數時間
  - 顯示日期、時間、地點

- **本週賽程快覽**
  - 當週所有場次的對戰組合（小卡片格式）
  - 點擊可跳至「本週賽程」完整頁面

- **積分榜快覽**
  - 六隊目前排名、勝率、近況 badge（連勝/連敗）

- **最新公告**
  - 最近一則公告標題與摘要

---

### 📅 本週賽程

**目的：** 呈現當週每場比賽的完整資訊，包含工作人員分配。

**內容區塊：**

- **賽事標頭**
  - 屆次、週次、比賽日期、地點

- **場次列表**（共6場）
  - 每場卡片包含：
    - 場次編號
    - 主場 vs 客場（隊伍顏色 + 名稱）
    - 比賽時間
  - 可展開查看工作人員：
    - 裁判名單（含隊伍標示）
    - 場務名單
    - 攝影負責人
    - 器材負責人
    - 場控負責人

- **週次切換**
  - 可往前/往後瀏覽歷史週次賽程

---

#### 📋 輪值排班（整合於本週賽程）

**目的：** 取代現有 Google Sheet 輪值安排表，每週五由系統根據請假名單自動產生最佳輪值分配。

**流程說明：**
1. 各隊領隊於賽前在系統標注本週請假成員
2. 每位球員設定「可輪值的場次」（1～6場任選）
3. 系統根據累積輪值次數由少到多，自動分配裁判、場務、攝影、器材、數據等職位
4. 系統略過已請假成員
5. 結果呈現於本週賽程頁面的工作人員欄位

**請假標注顯示：**
- 本週出席人數總計
- 各隊出席 / 請假名單一覽

**職責分類：**
- 裁判（多人）
- 場務（多人）
- 攝影（一人）
- 器材（一人）
- 數據記錄（一人，記錄者獲得積分）
- 場控（一人）

> **資料來源對應：** 輪值安排表「請假名單」分頁

---

### 🏆 積分榜

**目的：** 即時呈現各隊排名與戰績，讓成員掌握聯盟現況。

**內容區塊：**

- **排名總表**
  - 欄位：名次、隊伍、勝場、敗場、勝率、近況
  - 近況以 badge 標示（連勝綠色、連敗紅色）
  - 依勝率排序

- **對戰積分矩陣**
  - 各隊互相對戰的勝負差表格
  - 對應 Google Sheet 現有的積分矩陣
  - 正值（綠色）/ 負值（紅色）視覺化

- **賽季進度**
  - 進度條：已完成週次 / 總週次

> **資料來源對應：** Google Sheet「本周賽程」分頁右側戰績表與矩陣

---

### 🥇 積分龍虎榜

**目的：** 呈現每位球員的活躍度積分排名，作為下賽季選秀順位的依據，同時讓所有成員清楚了解自己目前的積分組成與排名狀態。

**內容區塊：**

- **排名總表**
  - 欄位：排名、球員姓名、所屬隊伍、總積分、出席次數、輪值次數、名次加分
  - 同分時依規則排序（出席率 → 輪值裁判次數 → 入盟年資 → 年長者）
  - **前36名**以視覺方式醒目標示（一般選秀資格線）
  - **前12名**另以特殊標示區分（全明星資格線）
  - 第37名以後標示為「新秀選秀名單」

- **積分組成明細**（可展開每位球員）
  - 出席次數 × 1 點
  - 輪值次數（裁判/場務）× 1 點
  - 名次加分（冠軍/亞軍/季軍 +2、第四/第五 +1）
  - 顯示總計

- **球員分類標籤**
  - 🌟 全明星（前12名）
  - ✅ 一般好手（第13～36名）
  - 🆕 新秀（第37名以後，或出席率未達門檻）

- **選秀資格說明連結**
  - 一鍵連結至公告欄中的選秀規則公告
  - 簡短說明文字：「積分將決定下賽季選秀順位，詳細規則請見公告」

- **篩選 / 搜尋**
  - 依隊伍篩選
  - 搜尋球員姓名
  - 切換顯示「本賽季」/ 歷史賽季龍虎榜

**積分計算規則說明區塊**（可收合）
| 項目 | 積分 |
|------|------|
| 出席一次 | +1 |
| 擔任輪值（裁判/場務）一次 | +1 |
| 賽季冠軍隊成員 | +2 |
| 賽季亞軍隊成員 | +2 |
| 賽季季軍隊成員 | +2 |
| 賽季第四名隊成員 | +1 |
| 賽季第五名隊成員 | +1 |

> **注意：** 名次加分於賽季結束後一次性計入，賽季進行中僅顯示出席與輪值累計。

> **資料來源對應：** 工作人員出勤表（輪值次數）+ 賽程出席紀錄 + 最終名次

---

### 👷 工作人員出勤

**目的：** 透明化每位成員的出勤紀錄，輔助管理員進行排班。

**內容區塊：**

- **本週應優先出勤名單**（最重要區塊）
  - 依職責分列：裁判、場務、攝影、器材
  - 標示出勤次數最少者為優先人選

- **全員出勤統計排行**
  - 依職責分頁籤（裁判 / 場務 / 攝影 / 器材）
  - 每人顯示：姓名、所屬隊伍、累積出勤次數
  - 依次數由少到多排列

- **依隊伍篩選**
  - 可篩選特定隊伍成員的出勤狀況

> **資料來源對應：** Google Sheet「本周賽程」分頁右下角出勤統計區塊

---

### 📜 歷史賽季

**目的：** 保存每屆聯盟的賽事紀錄，讓成員可以回顧歷史。

**內容區塊：**

- **屆次選擇器**
  - 下拉選單或分頁切換（第23屆、第24屆、第25屆…）

- **該屆最終排名**
  - 冠軍至末位的最終積分榜

- **歷屆冠軍紀錄**
  - 視覺化呈現歷屆冠軍隊伍

- **該屆完整賽程紀錄**
  - 可依週次瀏覽所有場次結果

> **注意：** 歷史資料需由管理員人工匯入，建議從第24屆開始建立。

---

### 📊 數據統計

**目的：** 整合賽後人工紀錄的個人數據，提供本季排行榜與跨賽季生涯數據，讓球員追蹤自身成長、社群感受聯盟歷史。

> **重要說明：** 數據紀錄者本身亦可獲得積分，此機制需與積分龍虎榜資料連動。

---

#### 本季排行榜

- **得分排行**：場均得分 Top 20
- **籃板排行**：場均總籃板（進攻 + 防守）Top 20
- **助攻排行**：場均助攻 Top 20
- **阻攻排行**：場均阻攻 Top 20
- **抄截排行**：場均抄截 Top 20
- **綜合效率排行**：EFF 與 GmSc 指數 Top 20
- **投籃命中率排行**：兩分球 % / 三分球 % / 罰球 % Top 20

每項排行榜顯示欄位：名次、球員姓名、所屬隊伍、數值

---

#### 單場個人數據記錄格式

每場比賽由記錄人賽後輸入，每位球員包含：
- 兩分球（命中 / 未命中）
- 三分球（命中 / 未命中）
- 罰球（命中 / 未命中）
- 得分、進攻籃板、防守籃板、總籃板
- 助攻、阻攻、抄截、失誤、犯規
- 是否出賽（出賽名單確認）

記錄人欄位：顯示本場記錄者姓名（連動積分加分）

---

#### 🏛️ 生涯數據榜 ／ 名人堂

> 資料來源：第11屆～第19屆、第24屆歷史數據

**名人堂（平均數據 Top 20）**
- 場均得分 / 場均籃板 / 場均助攻 / 場均阻攻 / 場均抄截
- 兩分球命中率 / 三分球命中率 / 罰球命中率
- EFF 效率值 / GmSc 比賽分數 / TS% 真實命中率
- TOV% 失誤率

**生涯累計數據榜（總數 Top 20）**
- 總得分 / 總籃板 / 總助攻 / 總阻攻 / 總抄截
- 總失誤 / 總犯規 / 總出賽場次
- 總兩分 / 總三分 / 總罰球
- 總進攻籃板 / 總防守籃板

**鐵人榜（出手總數 Top 20）**
- 總兩分出手鐵 / 總三分出手鐵 / 總罰球出手鐵 / 三鐵霸主

**奪冠榜**
- 歷屆冠軍次數排行

> **注意：** 生涯數據榜目前涵蓋屆次為第11～19屆、第24屆，未來每屆結束後自動累計。

### 👷 工作人員出勤

**目的：** 取代 LINE 群組的重要行政通知，集中呈現正式公告。

**內容區塊：**

- **公告列表**
  - 每則公告顯示：標題、日期、摘要
  - 置頂公告以標示區分

- **公告詳細頁**
  - 點擊後展開完整公告內容

- **公告分類**（可選）
  - 場地異動、規則更新、活動公告

## 四、頁面開發優先順序

| 優先級 | 頁面／模組 | 理由 |
|:---:|------|------|
| 🔴 P1 | 本週賽程 | 每週必看，直接取代 Google Sheet 核心功能 |
| 🔴 P1 | 積分榜 | 賽後立即查閱，高流量頁面 |
| 🔴 P1 | 積分龍虎榜 | 涉及選秀制度，成員高度關注自身排名 |
| 🔴 P1 | 輪值排班 | 每週五必須產出，取代 Sheet 自動分配邏輯 |
| 🟠 P2 | 數據統計－本季排行榜 | 球員關注個人數據，與積分連動 |
| 🟠 P2 | 工作人員出勤 | 管理員週週需要，與輪值連動 |
| 🟠 P2 | 首頁 | 入口頁，依賴 P1 資料完成後製作 |
| 🟡 P3 | 歷史賽季 | 重要但不緊急，資料需補建 |
| 🟡 P3 | 公告欄 | 選秀規則連結的必要依附頁面 |
| 🟡 P3 | 生涯數據榜／名人堂 | 資料已有，整合後即可呈現 |
| 🟢 P4 | 鐵人榜／奪冠榜 | 歷史彩蛋功能，第二階段迭代 |

## 五、各頁面所需資料欄位整理

### 比賽（Match）
- 屆次、週次、場次編號
- 主場隊伍 / 客場隊伍、比賽日期、時間、地點
- 比賽結果（比分）—— 管理員賽後輸入

### 個人單場數據（PlayerGameStat）
- 比賽關聯、球員關聯
- 兩分球命中 / 未命中、三分球命中 / 未命中、罰球命中 / 未命中
- 得分、進攻籃板、防守籃板、助攻、阻攻、抄截、失誤、犯規
- 是否出賽
- 數據記錄者（連動積分）

### 工作人員分配（StaffAssignment）
- 場次關聯、人員姓名、職責、所屬隊伍
- 是否為本週排班（自動產生 or 手動調整）

### 請假紀錄（LeaveRecord）
- 球員關聯、週次、請假 / 出席
- 可輪值場次（第1～6場，布林值）

### 輪值次數（RotationCount）
- 球員關聯、職責類型（裁判 / 場務 / 攝影 / 器材 / 數據）
- 本賽季累計次數（用於自動排班排序）

### 隊伍（Team）
- 隊伍名稱（顏色）、本賽季成員

### 積分龍虎榜（PlayerScore）
- 球員關聯、賽季關聯
- 出席次數、輪值次數、名次加分、總積分
- 球員類型（全明星 / 一般好手 / 新秀）
- 入盟年資、出生年（同分排序用）

### 生涯數據（CareerStat）
- 球員關聯、跨賽季累計各項數值

### 公告（Announcement）
- 標題、內容、發布日期、分類、是否置頂、外部連結（選秀公告用）

## 六、設計建議

- **色系**：以六隊顏色（紅、黑、藍、綠、黃、白）作為視覺語言的基礎，搭配深色背景呈現運動感
- **排版**：手機優先（Mobile First），卡片式佈局
- **字體**：中文使用粗體無襯線（如 Noto Sans TC），標題搭配英文運動風字型
- **動態效果**：排名變動、連勝/連敗 badge 加入動態提示
